IMAGE DISPLAY LOGIC:
───────────────────
```javascript
// Check if driver has an image
const hasImage = driver.imageUrl && driver.imageUrl.trim() !== '';

// Display image or placeholder
const imageSection = hasImage ? `
    <div class="driver-image-container">
        <img src="${driver.imageUrl}" alt="Driver Photo" 
             class="driver-thumbnail" 
             onclick="viewDriverImage('${driver.imageUrl}')">
        <button class="view-image-btn" 
                onclick="viewDriverImage('${driver.imageUrl}')">
            <i class="fas fa-eye"></i>
        </button>
    </div>
    <div class="image-actions">
        <button class="btn btn-sm btn-primary" 
                onclick="viewDriverImage('${driver.imageUrl}')">
            <i class="fas fa-eye"></i> View Image
        </button>
    </div>
` : `<div class="no-image-placeholder">No photo available</div>`;
```

MODAL FUNCTIONALITY:
───────────────────
```javascript
function viewDriverImage(imageUrl) {
    const modal = document.getElementById('imageModal');
    const modalImage = document.getElementById('modalImage');
    
    modalImage.src = imageUrl;
    modal.style.display = 'flex';
    document.body.style.overflow = 'hidden'; // Prevent scroll
}

function closeImageModal() {
    const modal = document.getElementById('imageModal');
    modal.style.display = 'none';
    document.body.style.overflow = 'auto'; // Restore scroll
}
```

EVENT HANDLERS:
──────────────
```javascript
// Close modal when clicking outside
document.getElementById('imageModal').addEventListener('click', function(e) {
    if (e.target === this) {
        closeImageModal();
    }
});

// Close modal with Escape key
document.addEventListener('keydown', function(e) {
    if (e.key === 'Escape') {
        closeImageModal();
    }
});
```

🐛 TROUBLESHOOTING:
================================================================================

ISSUE: Image not displaying
SOLUTION:
  → Check if driver has imageUrl
  → Verify image URL is valid
  → Check browser console for errors
  → Try refreshing the page

ISSUE: Modal not opening
SOLUTION:
  → Check if imageUrl is properly escaped
  → Verify JavaScript is loaded
  → Check browser console for errors
  → Try clicking the button instead of thumbnail

ISSUE: Modal not closing
SOLUTION:
  → Try clicking outside the image
  → Press Escape key
  → Click the close button
  → Refresh the page if stuck

ISSUE: Image too small/large
SOLUTION:
  → Modal automatically sizes to screen
  → Browser handles zoom functionality
  → Image maintains aspect ratio
  → Responsive design adapts

ISSUE: Placeholder showing instead of image
SOLUTION:
  → Check if imageUrl exists in driver data
  → Verify imageUrl is not empty string
  → Check if image failed to load
  → Verify Cloudinary configuration
