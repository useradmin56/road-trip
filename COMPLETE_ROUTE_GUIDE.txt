📋 STEP-BY-STEP INSTRUCTIONS:
================================================================================

STEP 1: SET YOUR STARTING POINT
───────────────────────────────
Go to: "Route & Stops" tab

1. Enter "Starting Point" name (e.g., "My Home - Delhi")

2. Add coordinates (Choose one method):

   METHOD A: Use Current Location
   ──────────────────────────────
   • Be at your starting location
   • Click "Use Current Location" button (under Start fields)
   • Coordinates auto-fill! ✅

   METHOD B: Enter Manually
   ────────────────────────
   • Enter Start Latitude: e.g., 28.6139
   • Enter Start Longitude: e.g., 77.2090

   METHOD C: Copy from Google Maps
   ───────────────────────────────
   • Open Google Maps
   • Right-click on your starting point
   • Click coordinates to copy
   • Paste: First = Lat, Second = Lng

3. Enter Start Date (optional)

STEP 2: SET YOUR DESTINATION
────────────────────────────
1. Enter "Destination" name (e.g., "Hotel in Jaipur")

2. Add coordinates (Choose one method):

   METHOD A: Use Current Location (if you're there)
   ────────────────────────────────────────────────
   • Click "Use Current Location" button (under Destination fields)
   • Coordinates auto-fill! ✅

   METHOD B: Enter Manually
   ────────────────────────
   • Enter Destination Latitude: e.g., 26.9124
   • Enter Destination Longitude: e.g., 75.7873

   METHOD C: Copy from Google Maps
   ───────────────────────────────
   • Open Google Maps
   • Right-click on destination
   • Click coordinates to copy
   • Paste into Lat/Lng fields

3. Enter End Date (optional)

4. Click "Save Route with Coordinates" ✅

   You'll see: ✅ Route information saved successfully!

STEP 3: ADD INTERMEDIATE STOPS
──────────────────────────────
Scroll down to "Stops" section

For each stop:
1. Enter Stop Name (e.g., "Gas Station", "Lunch Break")
2. Enter Location (optional description)
3. Add Latitude and Longitude:
   • Click "Use My Current Location" (if you're there)
   • OR enter manually
   • OR copy from Google Maps
4. Enter Arrival/Departure times (optional)
5. Add notes (optional)
6. Click "Add Stop" ✅

Repeat for all stops!

STEP 4: VIEW COMPLETE ROUTE
───────────────────────────
After saving start, stops, and end:

1. Scroll up in "Route & Stops" tab
2. See "Complete Route Summary" card:
   • Start: Your starting point name
   • Destination: Your end point name
   • Total Stops: Number of stops
   • Total Route Distance: XX.XX km (AUTO-CALCULATED!)

3. Go to "Live Location" tab
4. See your complete route on map:
   • Blue marker "START" at beginning
   • Green numbered markers "1, 2, 3..." for stops
   • Red marker "END" at destination
   • Green dashed line connecting all points
   • "Route Information" shows total distance

STEP 5: EXPORT YOUR ROUTE (Optional)
────────────────────────────────────
• Go to "Live Location" tab
• Find "Route Information" panel
• Click "Export Route Data"
• Downloads complete route as JSON file
• Includes all points, coordinates, distances

📏 DISTANCE CALCULATION:
================================================================================

HOW IT WORKS:
────────────
The system calculates distance between consecutive points:

START (28.6139, 77.2090)
  ↓ Calculate distance using Haversine formula
STOP 1 (27.9833, 76.3833)
  ↓ Calculate distance
STOP 2 (27.3833, 75.9667)
  ↓ Calculate distance
END (26.9124, 75.7873)

TOTAL = Sum of all segments

FORMULA: Haversine (great-circle distance on a sphere)
ACCURACY: Within 0.5% for straight-line distance
NOTE: Actual road distance is typically 10-15% more

EXAMPLE:
       Calculated: 265.5 km (straight line)
       Actual Road: ~300 km (with curves)
       Difference: ~35 km (13%)

🔧 TROUBLESHOOTING:
================================================================================

ISSUE: Route not showing on map
SOLUTION:
  ✓ Make sure you added coordinates (lat/lng) for start
  ✓ Add at least one stop with coordinates OR end point
  ✓ Go to "Live Location" tab to see map
  ✓ Refresh page if needed

ISSUE: Distance shows 0 km
SOLUTION:
  ✓ Check that all points have coordinates
  ✓ Make sure coordinates are valid numbers
  ✓ Re-save route by clicking "Save Route" button

ISSUE: "Use Current Location" not working
SOLUTION:
  ✓ Click "Enable Location Permission" first
  ✓ Allow location access in browser
  ✓ Make sure you're outdoors for GPS signal
  ✓ Check device location services enabled

ISSUE: Markers not showing correct colors
SOLUTION:
  ✓ START should be blue
  ✓ Numbers (stops) should be green
  ✓ END should be red
  ✓ If wrong, try "Clear Route" and re-add

ISSUE: Wrong coordinates entered
SOLUTION:
  ✓ Go back to "Route & Stops" tab
  ✓ Correct the latitude/longitude
  ✓ Click "Save Route" again
  ✓ Map updates automatically
